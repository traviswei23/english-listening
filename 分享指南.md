# 英语听力生成器 - 分享指南

## 🥇 方案一：GitHub Pages 部署（推荐 · 免费 · 最简单）

这是最推荐的方式！部署后获得一个永久网址，别人打开浏览器就能用，**不需要安装任何东西**。

### 你只需要操作一次（5分钟）：

**步骤 1**：打开 https://github.com ，注册/登录账号

**步骤 2**：点击右上角 **+ → New repository**
- Repository name 填：`english-listening`（或任意名）
- 选择 **Public**
- 勾选 ✅ **Add a README file**
- 点击 **Create repository**

**步骤 3**：进入刚创建的仓库，点击 **Add file → Upload files**
- 把 `D:\听力转换\public\standalone.html` 拖进去
- 重命名为 `index.html`（重要！）
- 点击 **Commit changes**

**步骤 4**：点击仓库顶部 **Settings** → 左侧 **Pages**
- Branch 选 `main`，点 **Save**
- 等待 1-2 分钟，页面刷新后你会看到一个网址：
  ```
  https://你的用户名.github.io/english-listening/
  ```

**完成！** 把这个网址发给任何人，他们打开就能用。

> ⚠️ 纯浏览器版用系统自带语音播放，不能下载 MP3。推荐用 **Edge 浏览器**打开，音质最好。

## 🥈 方案二：Railway 云部署（有 MP3 下载功能）

如果需要 **MP3 下载** 功能，可以部署完整服务器。Railway 有免费额度，每月 500 小时。

### 步骤：

**步骤 1**：把整个 `D:\听力转换` 文件夹上传到 GitHub 仓库
```bash
# 在 D:\听力转换 目录下运行：
git init
git add .
git commit -m "English listening TTS app"
git branch -M main
git remote add origin https://github.com/你的用户名/english-listening-tts.git
git push -u origin main
```

**步骤 2**：打开 https://railway.com ，用 GitHub 账号登录

**步骤 3**：点击 **New Project → Deploy from GitHub → 选择你的仓库**

**步骤 4**：Railway 自动检测 Node.js 项目并部署，1-2 分钟后生成公开网址。

**步骤 5**（重要）：进入项目 Settings，添加环境变量：
```
PORT=3000
```

完成！获得一个带 MP3 下载功能的永久网址。

> Railway 免费额度每月 500 小时，个人使用完全够。

## 🥉 方案三：直接发文件

把文件夹打包发给对方，他们本地运行：

```
1. 解压文件
2. 安装 Node.js: https://nodejs.org （LTS 版本）
3. 双击 start.bat
4. 浏览器打开 http://localhost:3000
```

适合发给会用电脑的人。

- **日常自己用**：双击 `start.bat` 本地运行（有 MP3 下载）
